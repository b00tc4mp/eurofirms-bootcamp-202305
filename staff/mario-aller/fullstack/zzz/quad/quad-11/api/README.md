# api

# Enviroment Variables

API_PORT = port of API

JWT_SECRET = secret string for jsonwebtoken

MONGOOSE_URL = mongodb address

## Modules to be installed previosly
- cors
- dotenv
- express
- jsonwebtoken
- mongoose

## To execute the api
Run
- place in api root directory ~/quad/api
- npm start

Debug
- place in api root directory ~/quad/api
- npm run debug
